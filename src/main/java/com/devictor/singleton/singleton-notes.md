
# Design Patterns

## Singleton

### Conceitos:

- Uma única instância

- Atributo static

- Método Lazy

- Multithread

- Enum

- Subclasses

---

### Vantagens

- Forma simples e direta de ter uma única instancia de um objeto.

---

### Desvantagens

- Confundir o uso com o de uma variável global.
